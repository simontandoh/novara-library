# Testimonials

## Variants

- **TestimonialsMarquee** - auto-scrolling card row, pauses on hover at edges via gradient mask. Use for trade, beauty, consultant, ecommerce, restaurant. Works best with 6+ testimonials.
- **TestimonialsEditorial** - single large pull-quote with attribution. Use for editorial, luxury, photography, hospitality. Pass one item, the rest are ignored.

## Best theme pairings

- TestimonialsMarquee + tradeTheme
- TestimonialsMarquee + luxuryTheme
- TestimonialsEditorial + editorialTheme
- TestimonialsEditorial + luxuryTheme

## Props

See `Testimonials.types.ts` - `TestimonialsProps`, `TestimonialItem`.

## Note

Marquee duration is fixed at 60s for 6+ items. Adjust manually in the component if you need slower/faster.
