# Hero

## Variants

- **HeroMinimal** — text-led, no image. Use for consultants, luxury brands, restrained editorial.
- **HeroCinematic** — full-bleed image with overlay. Use for editorial, photography, restaurant, luxury hospitality.

## Best theme pairings

- HeroMinimal + luxuryTheme
- HeroMinimal + editorialTheme (light)
- HeroCinematic + editorialTheme
- HeroCinematic + luxuryTheme
